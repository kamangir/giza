\marginpar{\keyword{--<keyword> <value>}}
\keyword{argparse}~\cite{argparse}, \keyword{click}~\cite{click}, \keyword{fire}~\cite{python_fire}, and many other command line parsers support the \keyword{--<keyword> <value>} convention. By ending calls to Python with \keyword{"\${@:<number>}"} these arguments are captured. Here is an example,
%
\begin{verbatim}
@message submit \
    [--data <data>] \
    [--filename <filename>] \
    [--recipient <host_1,host_2>] \
    [--subject <subject>]
\end{verbatim}
