\marginpar{\keyword{\$<core>\_<suffix>}}
System state is carried in a set of environment variables that are generaly \keyword{export}ed during initialization and are used to harmonize paths and other machine--specific parameters to enable. Here are examples, 
%
\begin{itemize}
    \item{\keyword{\$<module>\_<type>\_name[\_prev[n]]}: selected objects.}
    \item{\keyword{\$<module>\_is\_<suffix>} machine state.}
    \item{\keyword{\$<module>\_path\_<suffix>}: module paths.}
\end{itemize}
%
These variables aree listed by \keyword{@env [<keyword>]}.
