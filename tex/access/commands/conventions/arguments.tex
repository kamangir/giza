\marginpar{\keyword{--<keyword> <value>}}
\emph{argparse}~\cite{argparse}, \emph{click}~\cite{click}, \emph{fire}~\cite{python_fire}, and many other command line parsers support the \keyword{--<keyword> <value>} convention. These arguments are captured in commands by ending calls to Python with \keyword{"\${@:<number>}"}.
