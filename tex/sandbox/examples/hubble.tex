\subsection{\texttt{hubble}}\label{hubble}

\texttt{hubble}~\cite{hubble} is a \keyword{callables} that \texttt{select}s, \texttt{list}s, and \texttt{download}s data from AWS Open Data Registry~\cite{aws_open_data}. 
%
\begin{verbatim}
hubble select [dataset] <hubble-dataset-name>
hubble select [object] <hubble-object-name>

hubble list [dataset] [.|<hubble-dataset-name>]
hubble list [object] [.|<hubble-object-name>]

hubble download \
    [~dryrun,filename=<filename>|all,~ingest,upload] \
    [.|<hubble-object-name>] \
    [.|<object-name>]
\end{verbatim}
