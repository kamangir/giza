\subsection{\texttt{@cache}}
\label{cache}

The cache is a keyword-value dictionary available on every \keyword{machine} for reading, writing, and searching, that is enabled either through a SQL database~\footabcli{abcli/plugins/tags/functions.py} or a tool such as \texttt{mlflow}~\footurl{https://mlflow.org/}.
%
\begin{verbatim}
value=$($@cache read <keyword>)

@cache write <keyword> <value>

value=@cache search <options>
\end{verbatim}
%
When used for \keyword{objects}, \texttt{@cache} provides a \keyword2{tagging}{tag} mechanism,
%
\begin{verbatim}
@cache read <object-name>.<keyword>

@cache write <object-name>.<keyword> <value>

@cache clone <object-1> <object-2>
\end{verbatim}
